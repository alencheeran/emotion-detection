#  Emotion Detection Model

This project is an **Emotion Detection System** that recognizes human emotions (like Happy, Sad, Angry, Surprise, Neutral, etc.) from facial expressions using **Deep Learning** and **Computer Vision** techniques.

---

##  Features

- Real-time emotion detection via webcam  
- Supports multiple emotion categories  
- Works on both images and video streams  
- Lightweight and easy to integrate with other applications  
- Can be used for sentiment analysis, feedback systems, or interactive AI projects
